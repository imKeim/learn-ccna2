С первого взгляда кажется, что в конфигурации статических маршрутов нет каких либо сложных механизмов, в которых могут появиться неисправности. Тем не менее, если что-то может пойти не так - оно обязательно пойдет не так. 

В этом курсе будут описаны основные виды неисправностей, которые возникают при настройке статической маршрутизации, а также практические рекомендации по их эффективному устранению.
